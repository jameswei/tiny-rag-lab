### Design Decisions

- `dependency-groups` (PEP 735) used for dev deps rather than `[tool.uv.dev-dependencies]` — the PEP 735 form is standard and uv supports it.
- `uv.lock` added to `.gitignore`: the lockfile is generated by uv sync and contains resolved pinned versions. For a learning project without a CI pipeline yet, committing the lockfile is premature. This can be revisited when CI is added.
- `[project.scripts]` entry not added yet — CLI entry point is P1-T01's responsibility.
- `tests/__init__.py` included so pytest can resolve imports from the package without needing `src/` layout or `pythonpath` config.
