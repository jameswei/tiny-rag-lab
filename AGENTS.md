# Agent Entry Point

This repository is currently in proposal mode. It is intended to become a learning-first RAG engine/laboratory, but no implementation phase is active yet.

Before proposing or changing code, read:

1. `docs/proposal.md`
2. `docs/roadmap.md`
3. `docs/architecture.md`
4. `docs/agent-brainstorming.md`

Do not start implementation work until a concrete phase spec and taskboard exist. The first useful agent task is to critique, refine, and make the plan decision-complete.

Prefer the project philosophy from `tiny-duo-infer`: readable, teachable code over clever abstractions or production-grade completeness. Frameworks such as LangChain, LlamaIndex, Ragas, and vector databases may be useful references, but the core learning implementation should keep the mechanics visible.
