# tiny-rag-lab

`tiny-rag-lab` is a learning-first RAG engine/laboratory for understanding how classic retrieval-augmented generation works end to end.

It is a companion project to `tiny-duo-infer`: small, explicit, and built for study rather than production. The goal is to make the RAG lifecycle visible: document loading, chunking, metadata, embeddings, vector search, retrieval, prompt assembly, answer generation, citations, evaluation, and failure inspection.

No implementation phase is active yet. This repository currently contains the seed proposal for discussion with Codex, Claude Code, and other agents.
